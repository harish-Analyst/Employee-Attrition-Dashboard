# Employee-Attrition-Dashboard
![Employee Attrition Dashboard](dashboard_preview.png)

## 📌 Project Overview
This project analyzes employee attrition trends across departments, education levels, and years of service using Power BI.

## 🛠 Tools Used
- Power BI
- SQL
- Excel

## 📊 Key Insights
- Higher attrition observed in early tenure employees
- Sales department shows higher turnover
- R&D demonstrates stronger retention patterns

## 📁 Files Included
- Power BI dashboard (.pbix)
- HR dataset
- SQL queries
