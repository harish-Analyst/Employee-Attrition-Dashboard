# 📊 Employee Attrition Dashboard

![Dashboard Preview](dashboard_preview.png)

## 📌 Project Overview
This project analyzes employee attrition trends across departments, education levels, and years of service using Power BI.

The objective is to identify high-risk groups, understand retention patterns, and provide data-driven insights for HR decision-making.

---

## 🛠 Tools Used
- Power BI
- SQL
- Excel

---

## 📊 Key Insights
- Attrition is highest among early-tenure employees.
- Sales department shows higher turnover compared to other departments.
- R&D demonstrates stronger employee retention.
- Education levels vary significantly across departments.
